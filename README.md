# 🗣️ English Passive Voice Quiz

A modern, highly interactive, and customizable React application designed to help users master the English Passive Voice through engaging quizzes. Built with React and Vite, it fetches dynamic questions seamlessly.

![Hero Image](https://raw.githubusercontent.com/edumdp-dev/project-react-quiz/main/public/hero.png)

## ✨ Features

- **Dynamic Questions**: Seamlessly fetches and parses questions from a Google Sheets CSV using `papaparse`.
- **Modern UI/UX**: Designed with a sleek, Material Design 3-inspired component system, featuring soft drop shadows, rounded "pill" aesthetics, and vibrant interactive states.
- **Customizable Quiz Experience**: Choose the number of questions, toggle a strict countdown timer, and turn instant feedback (with justifications!) on or off.
- **Micro-Animations & Feedback**: Fluid transitions and immediate visual styling (success/error states) when answering questions.
- **Fully Responsive**: Optimised for mobile, tablet, and desktop layouts.

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/edumdp-dev/project-react-quiz.git
   cd project-react-quiz
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🛠️ Technologies Used

- **React 19** - UI Library
- **Vite 8** - Fast build tooling
- **PapaParse** - In-browser CSV parsing for grabbing questions
- **Vanilla CSS** - With a highly structured token-based design system tailored for modern aesthetics

## 🌍 Deployment

Easily deployable to Vercel. 
```bash
npx vercel --prod
```
